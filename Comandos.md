Commandos:
=========

X significa um número
argumentos entre ( ) são opcionais


Staff
-------

|Commando | Argumentos |  Descrição |
|:------:|:---------:|:--------------------------------------:|
|!clearchat | | Limpa o chat |
|!skip | | Pula a música atual |
|!roulette | | Abre uma roleta |
